# Create JD App

This project was created using [Create JD App](https://github.com/OrJDev/create-jd-app)

## Deploying To Vercel

### Building

```bash
  vercel build --prod
```
  
### Deploying 

```bash
vercel deploy --prod --prebuilt
```
  
### Enviroment Variables
  
- `ENABLE_VC_BUILD`=`1` .
- `DATABASE_URL`=`your database url` .
